This is a Python code to access register data or contact status from FPSigma Panasonic (FPG-C32T2H) PLC using MEWTOCOL-COM Protocol (Aromat Serial) and RS232 as an interface.

Requirements :
* Hardware :
1. FPSigma FPG-C32T2H Panasonic PLC
2. FPG-COM2 AFPG802 (RS232 Cassette for FPSigma)
3. RS232 to USB Converter
4. laptop/PC

* Software :
1. FPWIN Pro to download program to PLC
2. Python 2.7

You can also check the data from PLC using OPC Server by selecting "Aromat Serial" driver for FP series.
